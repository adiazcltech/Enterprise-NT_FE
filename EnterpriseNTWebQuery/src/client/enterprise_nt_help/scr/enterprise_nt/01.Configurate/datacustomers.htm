Datos del cliente

Datos del cliente es la herramienta que provee la aplicación para configurar la vista de la consulta web.

 A continuación, se muestra una ventana típica con sus partes: 


[Imagen de referencia - Datos del cliente.]

1. Logo.
Permite cargar una imagen para su aplicación en la parte superior izquierda,no se mostrara si tiene configurado un banner.

2. Banner. 
Permite cargar una imagen para su aplicación en la parte superior.

3. URl.  
Dirección URL del laboratorio o de la entidad.

4. Color. 
Permite cambiar el color de los botones de la aplicación, y el banner si no tiene configurado un banner.

5. Nombre del cliente. 
Indica el nombre que va tener su aplicación, si no se configura el logo ni el banner, se mostrara en la parte superior.
6. Información.   
Información para mostrar en el pie de página.
 
7. Información adicional.
Información adicional para mostrar en pie de página.
 
8. Botón guardar.
Este botón permite guardar los cambios y refrescar la web con los cambios configurados.

[Imagen de referencia - Configurado en datos del cliente.]

9. Menu.
 Pestaña oculta que contiene el menu, solo los usuarios administrador puede acceder a la configuración del sistema y ver la pestaña.
