Opciones

Opciones es la herramienta que provee la aplicación para  habilitar o deshabilitar funcionalidades en resultados y en el login.

A continuación, se muestra una ventana típica con sus partes: 


[Imagen de referencia -Opciones.]

1. Histórico. 
Usa el control para habilitar o deshabilitar el botón de historico en resultados.  

2. Gráfico histórico .
 Usa el control para habilitar o deshabilitar la gráfica de historico en resultados.  

3. Histórico condensado.
 Usa el control para habilitar o deshabilitar el reporte de históricos en PDF.  

4. Login con captcha.
 Usa el control para habilitar o deshabilitar la funcionalidad captchan al autenticarse, al habilitar esta opción debe ingresar la llave de google para usar el captchan, con lo que se generaría una imagen similar a la siguiente:

[Imagen de referencia - captchan.]

5. Cambio de contraseña.
 Usa el control para habilitar o deshabilitar el cambio de contraseña a usuarios de tipo de medico,paciente y cliente. si habilito la opción se mostraria el cambio de contraseña en el login y al dar click se generaria una imagen similar a la siguiente:

 

[Imagen de referencia - cambio de contraseña.]

Seleccione el tipo de usuario y digite el usuario, si el usuario corresponde se enviara un correo electronico para restablecer la contraseña.

6. Saldo pendiente.
 Usa el control para habilitar o deshabilitar saldo pendiente, si habilito esta opción cuando un tipo de usuario no administrador consulta los exámenes de una orden, y esta orden posee un saldo pendiente no pódra ver dichos exámenes.

7. Busqueda por orden.
 Usa el control para habilitar o deshabilitar busqueda por orden en resultados, si habilito esta opción en el modulo de resultados se mostrara un tab que le permitira búsqueda por número de orden específico, se generaria una imagen similar a la siguiente:

 [Imagen de referencia - cambio de contraseña.]

8. Mostrar confidenciales.
 Usa el control para mostrar los resultados de los exámenes que sean marcados como confidenciales en LIS. se mostrara el exámen pero el resultado aparecerá como restringido.

9. Tipo de bloqueo de pánico.
Usa el control para configurar como se mostraran los pánicos en un examen en el módulo de resultados, puede seleccionar una de estas tres opciones.

  • No bloquea: muestra los exámenes sin importar si es patológico.
  • Bloquea orden: si un examen es patologico no muestra los exámenes.
  • Bloquea examen: no muestra el examen que sea patológico.

10. Mostrar resultados.
 Usa el control configurar como se mostraran los exámanes de en el módulo de resultados, puede seleccionar una de estas dos opciones.

  • Ingresados: muestra todos los exámenes de la orden.
  • Verificados: muestra los exámenes de la orden que se encuentran verificados.

11. Botón guardar.
Este botón permite guardar los cambios y refrescar la web con los cambios configurados.

12. Menu.
Pestaña oculta que contiene el menu, solo los usuarios administrador puede acceder a la configuración del sistema y ver la pestaña.
