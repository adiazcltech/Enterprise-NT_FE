Tipo de usuario

Esta opción permite la creación de zonas para agrupar las diferentes sedes de LIS.

 A continuación, se muestra una ventana típica con sus partes: 


[Imagen de referencia - Tipo de usuario.]

1. Tipo de usuario.
 Representa el tipo de usuario con el cúal se va iniciar sesión, hay 4 tipo de usuarios fijos:

  • Medico.
  • Paciente.
  • Cliente.
  • Laboratorio.

2. Imagen. 
Este botón permite cargar la imagen que se visualizara en el login, de click y elija la imagen.

3. Cantidad de ordenes. 
Este control permite elegir el número de órdenes que puede consultar los diferente tipos de usuarios, exeptuando el administrador que puede ver todas las ordenes sin restricciones, el número visible de ordenes varia desde el valor mínimo que seria uno  y el máximo que son todas las ordenes.

4. Activar usuario. 
Este control permite habilitar y dehabilitar la visibilidad del tipo de usuario ocultando el repectivo icono ene módulo del login.

5. Información. 
Descripción del usuario activo.  

6 Botón guardar.
Este botón permite guardar y refrescar la web con los cambios, en el login se generaría una imagen similar a la siguiente:
 

  [Imagen de referencia - tipos de usuario configurados.]
 
7. Menu.
Pestaña oculta que contiene el menu, solo los usuarios administrador puede acceder a la configuración del sistema y ver la pestaña.
