Resultados

Resultados es la herramienta que provee la aplicación para realizar la consulta de ordenes a través de diferente filtros.

A continuación, se muestra una ventana típica con sus partes: 


[Imagen de referencia - Resultados.]

1. Fecha.
Permite buscar por la fecha de creación de la orden, mostrando las ordenes creadas en esa fecha. Se generaria una imagen similar a la siguiente:  

[Imagen de referencia - Filtro fecha.]

2. Historia.
 Permite buscar las ordenes de la historia digitada. se puede seleccionar el tipo de documento si se encuentra configurada. Se generaria una imagen similar a la siguiente:  

[Imagen de referencia - Filtro historia.]

3. Paciente.
 Permite buscar por los nombre o apellidos del paciente. se generaria una imagen similar a la siguiente:  

[Imagen de referencia - Filtro paciente.] 

4. Orden.
Permite buscar por la orden especifica. Se generaria una imagen similar a la siguiente:   

[Imagen de referencia - Filtro orden.]

5. Listado de ordenes.
 listado de ordenes  con su respectivo paciente que cumple con el filtro seleccionado.

6. Listado de pruebas.
 Es una lista que se muestra la información de cada examen realizado los cuales pueden ser seleccionados para realizar las actividades necesarias,y contienen un icono que muestra en que estado, acontinuación los estados se representa de la siguiente manera.

  • Ordenado:

  • Reportado:

         

  • Preliminar:

       

  • Impreso.

        

  • Validado.

       

7. Botón historico.
El botón se habilitara cuando seleccione uno o varios examenes y mostrara el historico del examen .
